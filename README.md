# REACT ISOMETRICBLOCKS

Quick isometric :100: on [codesandbox](https://codesandbox.io/s/scene-zdism)

Exports a `<Scene>` component that accepts `props`:

| prop | default |
| :---------- | :---: |
| of         |  array   |
| origin |  obj   |
| scale   |  func  |

Development oriented `props

| prop | default |
| :---------- | :---: |
| debug         |  false   
| plane |  undefined   |
| silhouette   |  false   |
